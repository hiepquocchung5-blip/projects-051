## MICELY BLUEPRINT
A bold, brutalist game development curriculum inspired by “make me nicely,” redesigned for a high-contrast, neon-infused interface.

### DESIGN OVERHAUL SUMMARY
- **High-Impact Typography:** Anton as the primary display font, paired with Inter for readability and JetBrains Mono for system-status metadata.
- **Brutalist Layout:** Neutral-950 background with heavy Neutral-800 borders and a radial dot grid main section for a technical, industrial feel.
- **Neon Green Accents:** Signature neon green (#00FF66) for active states, progress bars, hover highlights, and key callouts.
- **Visual Flair:** A subtle, outlined “100” decal floats in the background to reinforce the roadmap goal.
- **Status Bar Footer:** A persistent footer tracks Milestone progress with a stylized neon meter.
- **Upper-Case Aesthetic:** Navigation and metadata use uppercase styling with wide tracking for a strong brutalist voice.
